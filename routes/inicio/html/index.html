SatelitLeo: Plataforma Climática Satelital

SatelitLeo es una aplicación web orientada al análisis meteorológico y geoespacial en tiempo real. Su propósito es demostrar cómo la integración de datos satelitales de alta precisión puede emplearse para visualizar, analizar y comprender patrones climáticos de manera accesible y científica.

Nuestra plataforma muestra cómo podría desarrollarse una herramienta de este tipo: combinando datos de la API de la NASA con la tecnología meteorológica de Meteomatics para generar mapas, series temporales y análisis interactivos.

SatelitLeo utiliza dos endpoints clave de Meteomatics:

  • Time Series: permite obtener valores meteorológicos en puntos geográficos específicos a lo largo del tiempo. Ideal para análisis climáticos detallados, históricos o en tiempo real.
  • Grid: devuelve información meteorológica distribuida en una rejilla espacial, permitiendo generar mapas y representaciones geoespaciales completas de variables como temperatura, viento o radiación solar.

Objetivo

Demostrar cómo la integración de datos satelitales y meteorológicos puede usarse para construir herramientas interactivas de análisis climático y potencial energético a nivel global.
